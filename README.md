# AI Interior Design Agent

## Overview

Interior design recommendation system using:

- Catalog Search
- Budget Validation
- Layout Validation
- Guardrails
- Evaluation Harness

## Setup

Install dependencies:

pip install -r requirements.txt

## Run Application

streamlit run app.py

## Run Evaluation

python eval_harness.py

## Components

- app.py
- agent.py
- database.py
- catalog_tool.py
- budget_tool.py
- layout_tool.py
- eval_harness.py

## Constraints

- No structural advice
- No plumbing recommendations
- No electrical recommendations

## Supported

Living Room recommendations only
